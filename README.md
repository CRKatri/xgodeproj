# xgodeproj

## Description

parse xcodeproj/project.pbxproj

## Usage

### show all section name

```bash
$ xgodeproj show <your_project.pbxproj>
```

### show section [TODO]

```bash
$ xgodeproj show <your_project.pbxproj> --section <section name>
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/mpon/xgodeproj
```
